So I totally forgot to blog that I had refreshed the RC in SVN. This is a minor change to the locale functionality. You can now supply a locale like en_GB and NOT supply a resource bundle file for it. This is handy for folks who may want to change date formatting (I still day d/m/y is crazy) but not want to bother changing the locale strings.

Small Refresh

Sep 20

posted by Raymond Camden in BlogCFC | 0 Comments

There was a small refresh of the SVN tonight. This should be the last small update as I really want to get 5.5 wrapped. (I've got two huge Canvas and Soundings updates I'd like to get done before MAX.)

  • Official blog.cfc logo is now used in the admin. Joe isn't credited in the docs yet but will be.
  • Scott did the CSS updates for this.
  • William Haun wrote a pod that lists all pages for your site. This is cool for navigation. I did NOT include it in the layout.cfm file because - well - I'm a bit concerned about overwhelming the default layout. But it is there if you want it.
  • I made a rss pod named feed. The idea is you just change the RSS url and you can include the RSS feed of another blog on your blog.
  • Small fix in the sql server install script.

Ok - no more small changes. The next build will be a Release Candidate. I promise.
