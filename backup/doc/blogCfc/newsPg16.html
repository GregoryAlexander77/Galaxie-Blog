BlogCFC 5.5 Beta 1

Sep 5

posted by Raymond Camden in BlogCFC | 17 Comments

Yes, I said 5.5. It's funny - when you control a project you can pick any old version you want. I think my next release will be version Pi. Updates in this version are:

  • Slideshows can now have names and captions. This is done via a new administrator tool. This tool also lets you create slideshows and upload images. I forgot to update the Word doc to describe this though.
  • XML-RPC: It is there. It has NOT been checked for security issues, but it should be secure. In the readme.txt I said it wasn't secure, but that was just to cover my rear. It should be ok. It doesn't have real good fault support. I've tested with Ecto and it works fine for me, but Lola had issues on her Mac, so please try this out folks.
  • Small fixes and updates here and there.

Plans: File Manager that I mentioned before. Finish Render support. That should be it.
