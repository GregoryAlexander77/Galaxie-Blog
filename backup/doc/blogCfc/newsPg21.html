BlogCFC 5.1: Textblock Editing

Jul 3

posted by Raymond Camden in BlogCFC | 7 Comments

Textblocks are random, simple blocks of text. Each textblock has a label that should be unique. BlogCFC would be updated to let you edit these things. An API would be provided to let you use them in the site.

For example. Imagine you have a links pod. It contains links to your favorite sites. It would be nice if you could edit it via BlogCFC. In your link pod, to get the content, you would simply have:

<cfset content = application.blog.getTextBlock("My Links")>

BlogCFC actually ships with a "simple content editor", but I'd like to make it a bit more formal and place it in the admin as a real tool.

Thoughts?

BlogCFC 5.1: Simple Page Editing

Jul 3

posted by Raymond Camden in BlogCFC | 5 Comments

Imagine this. You would be able to use BlogCFC to create a simple set of pages. So for example, if I want to make an About Ray page, I'd go to a new pages action in the admin. I'd set the page title: "About Me". BlogCFC would auto-create a URL for it, About-Me. I'd enter my body content. I'd then have a new page at:

myblogroot/pages.cfm/About-Me

BlogCFC wouldn't auto-create links for the page, but I could do that myself. Other pages would use the same URL, just different path info at the end. Layout would be the same as the main blog pages of course.
