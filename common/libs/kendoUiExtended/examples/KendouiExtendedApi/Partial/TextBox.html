kendo.ui.ExtTextBox extends kendo.ui.Widget
Introduced in 2013 Q1
Description
The ExtTextBox widget displays a TextBox that applies the Kendo styles and supports a placeholder that works with HTML5 and non HTML 5 browsers.
Example


Configuration
placeholder : String
The watermark that is displayed when the input element is empty.
placeholderClass : String
The title to be displayed in the titlebar.
textboxClass : String
The css class to be applied to the parent div of the input element.
width : String
The width for the TextBox widget.
