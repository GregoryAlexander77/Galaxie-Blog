kendo.ui.ExtDropDownTreeView extends kendo.ui.Widget
Introduced in 2013 Q1
Description
The ExtDropDownTreeView widget displays a Kendo DropDownList that shows a Kendo TreeView instead of a list when clicked.
Example
Configuration
treeview : kendo.ui.TreeView
The configuration settings for the Kendo Grid.
Methods
dropDownList
Returns a reference to the Kendo DropDownList.
treeview
Returns a reference to the Kendo TreeView.
Events
select
Triggered when a node is being selected by the user.
